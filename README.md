# Portfolio

Este repositório contém o código fonte do meu portfólio pessoal, desenvolvido para apresentar meus projetos, experiências, habilidades e informações profissionais de maneira clara, moderna e responsiva.

## Finalidade

O principal objetivo deste projeto é servir como um site de portfólio online, permitindo que recrutadores, colegas de profissão e demais interessados conheçam um pouco mais sobre mim, minhas competências técnicas e os trabalhos que já realizei. Aqui, busco centralizar informações relevantes sobre minha trajetória e facilitar o contato para futuras oportunidades.

## Funcionalidades

- **Página Inicial:** Breve apresentação e destaque de habilidades principais.
- **Sobre Mim:** Informações sobre minha formação, trajetória, valores e objetivos profissionais.
- **Projetos:** Seção dedicada aos principais projetos desenvolvidos, com descrições, tecnologias utilizadas e links para repositórios ou demonstrações.
- **Habilidades:** Lista das stacks e ferramentas que domino, com ícones representativos.
- **Experiência Profissional:** Resumo das experiências relevantes, cargos ocupados e empresas.
- **Contato:** Formulário para contato direto ou informações para conexão via e-mail e redes sociais.
- **Layout Responsivo:** Interface adaptada para diferentes dispositivos (desktop, tablet e mobile).
- **Design Moderno:** Visual limpo, intuitivo e alinhado com as tendências atuais de UI/UX.

## Tecnologias Utilizadas
- HTML5, CSS3, JavaScript
- Frameworks/Libraries: React
- Deploy: Vercel
